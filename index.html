자기소개

대표 개발자가
직접 책임집니다

QuokkaGround(쿼카그라운드)는 대표 개발자 장태관이 직접 실무를 수행하는 전문 개발팀입니다. 2016년부터 풀스택 개발자로 활동하며 20개 이상의 서비스를 기획·개발·출시한 경험을 보유하고 있습니다.

Flutter 기반 iOS·Android 크로스플랫폼 앱을 주력으로, Node.js / Spring Boot 백엔드, React·Next.js 관리자 웹, AWS 인프라까지 핵심 개발자가 직접 설계하고 구현합니다. 중간 전달자 없이 빠른 의사결정이 가능하고, 필요 시 기획자·디자이너·서버 개발자와 협업하여 웹·앱·백엔드 전 영역에 대응합니다.

Wishket 등 국내 주요 플랫폼에서 스타트업·개인사업자·중소기업 클라이언트와 함께 서비스를 만들어왔습니다.

클라이언트 후기

함께한 분들의 이야기

Wishket 등에서 실제 프로젝트를 진행한 클라이언트분들의 피드백입니다. 개인정보 보호를 위해 익명으로 표기했습니다.

★★★★★

운영 중인 프로젝트의 API 유지보수를 맡겨드렸는데, 정해진 일정을 정확히 지켜주셨습니다. 질의응답이 빠르고, 중간중간 진행 상황도 자세히 공유해주셔서 안심하고 맡길 수 있었습니다.

조용하고 차분한 성격이면서도 끝까지 책임감 있게 임해주셨습니다. 매일 데일리 미팅에 빠짐없이 참여하며 상황을 명확히 공유해주셔서 협업이 원활했습니다. 작은 디테일의 버그까지 끝까지 수정·개선하려는 태도가 인상적이었습니다.
