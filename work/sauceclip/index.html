앱 개발

숏폼 영상 편집앱 SauceClip Flutter 개발 및 네이티브 영상편집 통합

FlutterDartKotlinSwiftMethod ChannelEvent Channel영상편집네이티브 플러그인

참여 기간 · 2023.08 ~ 2023.12

포트폴리오 소개

  • 숏폼 콘텐츠 제작을 위한 영상 편집 앱 'SauceClip'을 Flutter로 개발한 프로젝트입니다.
  • Flutter로 기본 제공되지 않는 영상편집 기능은 Kotlin(Android)·Swift(iOS)로 직접 개발해 Flutter 플러그인으로 병합하고, Method Channel·Event Channel로 앱과 통신하도록 구현했습니다.

작업 범위

  • Flutter 기반 영상편집 앱(iOS·Android) 개발
  • Kotlin(Android)·Swift(iOS)로 영상편집 기능 각각 개발
  • Native 기능을 Flutter 플러그인으로 병합해 기능 확장
  • Method Channel·Event Channel로 Flutter↔네이티브 통신 구현

주요 업무 및 기능

  • 앱: Flutter로 영상편집 앱 화면·흐름 개발(iOS·Android 동시)
  • 네이티브 편집: Kotlin·Swift로 영상편집 기능을 플랫폼별 구현
  • 통합: 네이티브 편집 기능을 Flutter 플러그인으로 병합
  • 브릿지: Method Channel(요청·응답)·Event Channel(스트림/진행상태)로 통신

주안점

  • Flutter로 안 되는 영상편집을 네이티브로 구현해 안정적으로 통합
  • Method/Event Channel 통신의 정확성과 성능(진행상태·대용량 데이터)
  • iOS·Android 양쪽에서 동일한 편집 경험 제공

문제점

  • 영상편집은 고성능·플랫폼 종속적인 처리가 필요해 Flutter만으로는 한계가 있었습니다.
  • 앱은 iOS·Android를 Flutter로 동시에 개발해야 했습니다.
  • 네이티브 편집 기능과 Flutter 앱이 안정적으로 통신·통합돼야 했습니다.

프로젝트 목표

  • Flutter로 영상편집 앱을 iOS·Android 동시 개발.
  • 영상편집 핵심 기능은 Kotlin·Swift 네이티브로 구현해 품질·성능 확보.
  • 네이티브 기능을 플러그인으로 병합하고 Method/Event Channel로 통합.

주안점

  • 네이티브↔Flutter 통신(Method/Event Channel)의 안정성.
  • 플랫폼별 영상편집 기능의 일관성.
  • 플러그인 구조로 기능 확장이 용이하도록 설계.

프로젝트 성과

Flutter 영상편집 앱 개발(iOS·Android)

Flutter로 숏폼 영상편집 앱을 iOS·Android 동시 개발해, 하나의 코드베이스로 양 플랫폼을 지원했습니다.

네이티브 영상편집 기능 개발

Flutter로 제공되지 않는 영상편집 기능을 Kotlin(Android)·Swift(iOS)로 각각 개발했습니다.

네이티브 기능 Flutter 플러그인 통합

네이티브 편집 기능을 Flutter 플러그인으로 병합하고, Method Channel·Event Channel로 앱과 통신하도록 구현했습니다.
