포트폴리오 소개

  • 사주명리 데이터를 기반으로 두 사람의 궁합·인연 가능성·연애 성향을 분석하는 AI 소개팅 서비스 '여운'을 기획부터 출시까지 직접 구축한 자체 서비스입니다.
  • 앱(React Native), 백엔드(Supabase·Firebase), 어드민·랜딩(React), AI 궁합 리포트(OpenAI·Claude), 정기구독·포인트 결제까지 전 영역을 직접 개발했습니다.

작업 범위

  • 서비스 기획·핵심 컨셉 설계, 초기 브랜딩·랜딩/소개 문구
  • 사주명리 기반 매칭 알고리즘 설계·개발
  • AI 궁합 리포트 구조 설계 및 프롬프트 엔지니어링(OpenAI API + Claude API)
  • React Native 모바일 앱(iOS·Android) 개발
  • Supabase + Firebase 백엔드 개발
  • React 어드민 관리자 웹·랜딩 사이트 개발
  • 정기구독·결제형 포인트 시스템 기획·개발
  • 앱스토어/플레이스토어 출시·운영

주요 업무 및 기능

  • 사주 매칭: 사주명리 데이터로 팔자·십신·궁합 조합을 분석하는 매칭 로직
  • AI 리포트: OpenAI·Claude로 재물운·직업운·인맥운 등 궁합 리포트 생성(구조 설계·프롬프트)
  • 앱: React Native로 사주 프로필·매칭·리포트 UI 구현
  • 백엔드/어드민: Supabase·Firebase 백엔드, React 어드민·랜딩
  • 결제: 정기구독·포인트 시스템

주안점

  • 사주명리 로직을 매칭·리포트로 연결하는 데이터 구조 설계
  • AI 리포트의 일관성·품질을 위한 프롬프트 엔지니어링
  • 앱-백엔드-어드민을 아우르는 풀스택 정합성

문제점

  • 사주 궁합은 '재미로 보는 것'에 머무는 경우가 많아, 실제 매칭·연애로 이어지는 구조가 필요했습니다.
  • 사주명리 로직을 매칭과 AI 리포트로 일관되게 연결해야 했습니다.
  • 자체 서비스로 앱·백엔드·어드민·결제까지 스스로 만들어 출시해야 했습니다.

프로젝트 목표

  • 사주명리 데이터 기반 매칭과 AI 궁합 리포트를 결합한 소개팅 서비스 MVP를 기획·출시.
  • 앱(React Native)·백엔드(Supabase·Firebase)·어드민(React)까지 풀스택으로 구축.
  • 정기구독·포인트 결제로 수익 구조 마련.

주안점

  • 사주명리 매칭 알고리즘과 AI 리포트의 연결·품질.
  • 풀스택 정합성(앱-백엔드-어드민).
  • 출시·운영과 초기 브랜딩.

프로젝트 성과

기획~출시 풀스택 단독 구축

서비스 기획부터 사주 매칭·AI 리포트·앱·백엔드·어드민·결제·출시까지 전 과정을 직접 설계·개발했습니다.

사주명리 + AI 궁합 리포트

사주명리 매칭 알고리즘을 설계하고, OpenAI·Claude로 재물운·직업운·인맥운 등 궁합 리포트를 생성하도록 구조를 설계했습니다.

앱스토어·플레이스토어 출시·운영

React Native 앱을 iOS·Android로 출시하고, 정기구독·포인트 결제와 함께 운영했습니다.
